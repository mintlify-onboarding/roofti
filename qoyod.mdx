In order to get started with our Qoyod integration, you will need to first get the API Key. After that, you will need to enter your API Key on the RootFi .

These steps to get the API Key are outlined in greater detail below.

  
---

### Get API Key

  
1. Login to your [Qoyod Dashboard](https://www.qoyod.com/ara/%D8%AA%D8%B3%D8%AC%D9%8A%D9%84-%D8%A7%D9%84%D8%AF%D8%AE%D9%88%D9%84/).

  
1. Navigate to General Settings under Setting on the sidebar. Click 'Generate New Key' and click 'Save'.

![](/images/d38c9ae-image.png)

1. Now copy the API Key and paste the API Key to the Rootfi SDK.

![](/images/a7684b0-image.png)
